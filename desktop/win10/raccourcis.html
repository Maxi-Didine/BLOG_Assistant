Bureautique - Windows 10

Les principaux raccourcis Windows 10

Introduction

Afin d’optimiser votre utilisation de Win10, je vais vous proposer une liste de petits raccourcis qui sont sacrément bien utiles. Je ne vous demande pas de tous les retenir, mais en utiliser un par-ci par-là, ça va vous faire économiser de précieuses minutes.
Un petit truc aussi : vous pouvez inscrire ceux qui vous intéressent sur des pense-bêtes, ça finira par rentrer dans votre mémoire sans effort.
Evidemment, cette liste n'est pas exhaustive.

Souris

Raccourcis Résultat
Clic avec molette de la souris sur lien hypertexte ouvrir dans un nouvel onglet
Ctrl + molette de la souris (dé)zoomer dans une page

Touches directes

Ici, il faut prêter attention au fait que tous les ordinateurs ne sont pas programmés de la même façon. Pour certains, il faudra appuyer sur la touche fn, pour d’autre sur le Maj (testez une fois et vous serez fixés) et pour d’autres la simple touche d’action sera suffisante.

Raccourcis Résultat
fn ou Maj + Home faire aller le curseur en début de ligne
fn ou Maj + End faire aller le curseur en fin de ligne
fn ou Maj + PgUp faire aller le curseur en début de page
fn ou Maj + PgDn faire aller le curseur en fin de page
num lk (dé)activer le pavé numérique (si vous en avez un)

Touche contrôle

Raccourcis Résultat
Ctrl + A tout sélectionner
Ctrl + C copier un élément sélectionné
Ctrl + X couper un élément sélectionné
Ctrl + V coller un élément sélectionné
Ctrl + Z annuler la dernière action
Ctrl + Y restaurer la dernière action
Ctrl + S sauvegarder un document
Ctrl + P lancer l’impression

Touche Windows

Raccourcis Résultat
Win (tout seul) ouvrir le menu Démarrer
Win + L verrouiller sa session
Win + D afficher le bureau
Win + S ouvrir la recherche
Win + flèches déplacer la fenêtre sur l’écran

Capture d’écran

Raccourcis Résultat
Prt Sc faire une capture d’écran totale
Win + Maj + S faire une capture d’écran partielle

Bureaux virtuels

Raccourcis Résultat
Win + Ctrl + D ajouter un bureau virtuel
Win + Ctrl + flèches gauche/droite basculer entre les bureaux virtuels

Sélection des dossiers

Raccourcis Résultat
Ctrl + clics sélectionner plusieurs dossiers
Maj + clics sélectionner tous les dossiers entre les 2 cliqués

Conclusion

Bon je l’avoue, cet article n’était pas super amusant, mais parfois une liste vaut milles mots.

Sandrine Hoeffler, le 30/07/2019
